Acessibilidade

Para auxiliar a navegação, foram inseridas algumas teclas de atalho (accesskey) em alguns links. São elas:

  • Alt+1 — Voltar para a página inicial
  • Alt+2 — Ir para página “Institucional”
  • Alt+3 — Ir para página “Publicações Oficiais”
  • Alt+4 — Ir para página “Transparência”
  • Alt+5 — Ir para página “Portal da Transparência”
  • Alt+6 — Ir para página “Contato”
  • Alt+A — Acessibilidade do Site
  • Alt+I — Inverter as cores do site
  • Alt+M — Visualizar mapa do site

**Observação:** o uso exato de Alt ou Alt+Shift varia conforme o navegador.
